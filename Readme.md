# 📊 Data Visualization using Seaborn

## 🚀 Project Overview
This project demonstrates various **data visualization techniques using Seaborn**, a powerful Python library built on top of Matplotlib.

The goal is to perform **Exploratory Data Analysis (EDA)** and extract meaningful insights from data using different types of plots.

- Pandas 🧮
